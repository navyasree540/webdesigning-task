Project


Title :Searching of Relevant Information over Encoded Data in Cloud

Description :
Cloud is used by many number of users to outsource their data. These data is traversed to the cloud in an encryption format to preserve the privacy of data. However, this practice of encryption makes the utilization of data difficult. Such as search of any part of data by the keyword is difficult in the encrypted data. Even though the data is searched and retrieved there is a lot of semantic disturbance. To overcome this problem we proposed ECSED, a semantic aware searching concept. In this model we used two cloud server, one of the data storing and return searched data to the user, second is to search for similarity between the data and query. This method uses tree-based approach to well document the index vectors.
